CSS Selectors

We will learn about CSS

1. Element Selectors in CSS targets the specific elements like different tags

  1. Class Selectors in CSS target particular IDs by using a symbol '.'.
  2. ID Selectors in CSS target particular class by using a symbol '#'.
  3. Attribute Selectors in CSS target particular attributes of the specific tags by using the symbol '[]'.
  4. Universal Selector targets each and everything in the web page.
